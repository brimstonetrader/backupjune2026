```
Currently, `BSIZE = 512` bytes (from `fs.h`).

## Calculation if BSIZE were 1024 bytes:

With 1024-byte blocks:

1. **Direct blocks**: `NDIRECT = 12` blocks
   - Data: `12 × 1024 = 12,288 bytes`

2. **Indirect block**: 1 block pointer in `addrs[NDIRECT]`
   - Each indirect block can hold: `1024 / 4 = 256` block addresses (since `sizeof(uint) = 4`)
   - Data: `256 × 1024 = 262,144 bytes`

3. **Total maximum file size**:
```
   MAXFILE = NDIRECT + NINDIRECT
           = 12 + 256
           = 268 blocks
   
   Max size = 268 × 1024 = 274,432 bytes = 268 KB
