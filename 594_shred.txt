When playing against suboptimal opponents in extensive form games, it can be useful to veer from equilibrium when doing so gives more of an advantage than the Nash does. There exists several algorithms that do this, while maintaining safety guarantees such that the expected value does not veer below equilibrium. These algorithms generally do not scale to large games, because of the large amount of possible strategies and outcomes that must be assessed. I have implemented and analyzed two algorithms from Safe Opponent Exploitation (Ganzfried, Sandholm): Risk What You’ve Won in Expectation (RWYWE), and Best Equilibrium Followed by Full Exploitation (BEFFE). We have made some efficiency adjustments to the former algorithm, using a limited search via Monte Carlo CFR to find a good, but not necessarily optimal strategy to compare to, and using a small transformer model to predict the opponent's strategy, instead of maintaining a table or assessing all possible strategies. These changes, and the following analysis and statistics, are the main contribution of this project. 

These changes allow the algorithm to run in games many orders of magnitude larger than those of the original algorithm, with only mild losses in performance. Also included are comparisons with other notably efficient algorithms, such as DREAM (a neural CFR variant) and Lazy CFR, as well as comparisons with other exploitation algorithms, as specified above. Some theoretical guarantees from the original paper remain, but many are now only practical heuristics. This project was implemented in OpenSpiel, a python framework for reinforcement learning in games.
